Detailed Description

Interface allowing custom behaviour of Mergers components.

Interface allowing custom behaviour of Mergers components - unpacking the object into vector of objects, merging objects and providing timestamp of the object. Keep in mind, that you aside from implementing needed functions, one should activate them in MergerConfig.
